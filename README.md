# Kanban-board
 Another adventure of a software engineer
## Description
Well, this is the beginning of a long journey in learning frontend technologies. The first pet project will be Kanban.
Project stack:
- Core: React + TypeScript + Vite.
- Styles: Tailwind CSS.
- State Management: Zustand.
- Drag & Drop: @dnd-kit (core + sortable).
- Icons: lucide-react
